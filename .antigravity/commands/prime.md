# Context Loader - Priming Skill

This command is used at the start of any session to load project context and align the agent.

## Execution Steps

1. Read `ANTIGRAVITY.md` in the project root to understand architecture, conventions, and file structures.
2. Read `PRD.md` in the project root to understand requirements, features, and current roadmap.
3. Check the `reference/` directory for API, architecture, database, and testing specifications.
4. Scan the repository for active implementations.
