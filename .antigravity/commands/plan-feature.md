# Plan Feature Command

This command creates a detailed implementation plan before writing any code.

## Execution Steps

1. Analyze the feature request and identify the files that need to be created or modified.
2. Outline the step-by-step implementation plan including tests and validation checks.
3. Save the plan as a Markdown file in `.antigravity/plans/<feature-name>.md`.
4. Wait for user confirmation or feedback before starting the implementation.
