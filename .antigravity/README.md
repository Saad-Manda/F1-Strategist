# Antigravity Automation Environment

This directory contains the automation environment settings, commands, skills, and templates for this project.

## Directory Structure

- `commands/`: Markdown files that act as loaded skills or instructions for automation tasks (e.g., priming, feature planning, execution).
- `skills/`: Specialized automated skills for browsers, testing, and more.
- `templates/`: Templates for creating new project files (e.g. PRD, reference documentation).
- `plans/`: Implementation plans created before developing new features.
