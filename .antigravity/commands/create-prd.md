# Create PRD Command

This command scaffolds a new Product Requirements Document using the `.antigravity/templates/PRD-template.md` as a starting point.

## Execution Steps

1. Read `.antigravity/templates/PRD-template.md`.
2. Gather project context from user request or existing ideas (e.g. `docs/idea.md`).
3. Write/update `PRD.md` in the project root with detailed requirements.
